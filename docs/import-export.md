# [実験的実装] インポート・エクスポート
 
「Try it」画面から Workbench の状態をエクスポートできます。また「Home」画面では「IMPORT」タブを選択することでインポートできます。

また、インポート時には基本となる画像の URL を差し替えることもできます。

ただし、現状では仕様が固まっていないのでインポートできなくなる可能性が高いです。
